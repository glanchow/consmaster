<a name="installation"/>
2. Installation
---------------

You will need PySide and SqlAlchemy

    $ sudo apt-get install python-pyside
    $ sudo apt-get install python-sqlalchemy
    $ git clone https://github.com/glanchow/consmaster.git

Start the client

    $ cd consmaster/src/
    $ ./consmaster.py

Start the server

    $ ./consmaster-server.py
